# üçø Friendflix Full-Stack Application üé¨

Friendflix is a web platform that allows users to share movie recommendations with friends. It helps users decide which movie to watch, based on suggestions from friends or popular picks within the app. The platform facilitates the exchange of recommendations and simplifies decision-making.

## Preview

Live demo: https://lara.v1-22.proyectosdwa.es/

### Features

-   User Registration and Login: Users can create an account and log in to the platform.
-   User Management: Modify and delete registered users.
-   Movie Recommendations: View movie recommendations based on popular picks or friend suggestions.
-   Favorites Management: Add or remove movies from the user's favorites list.
-   Search Functionality: Search for movies by title.
-   Friends' Favorites: View movies that your friends have saved as favorites.
-   External Links in Footer: Links to external resources in the footer section.

#### Technologies Used

üî∏ Front-End:

-   HTML5 and CSS3 for structuring and styling the app.
-   React.js (https://reactjs.org/) for building the user interface.
-   Inertia.js (https://inertiajs.com/) to seamlessly connect the front-end with the back-end without full page reloads.
-   Tailwind CSS (https://tailwindcss.com/) for rapid and responsive UI development.
-   JavaScript as the primary programming language.

üî∏ Back-End:

-   Laravel (https://laravel.com/): PHP framework used for server-side logic.
-   MySQL: Relational database for storing user data, movies, and recommendations.

üî∏ API:

-   The Movie Database (TMDb) (https://www.themoviedb.org/): Provides movie and series data used within the platform.

üî∏ Other Tools:

-   TablePlus (https://tableplus.com/): Database management tool.
-   Git: Version control.
-   Docker (https://www.docker.com/): Containerization of the application to streamline development and deployment.
-   Figma (https://www.figma.com/): Design tool used for planning the user interface.
-   React-icons (https://react-icons.github.io/react-icons/): Icon library for React.
-   Axios (https://axios-http.com/): Library for making HTTP requests from the client side.
